​	chmod命令：

​	**change mod** 

```sh
chmod u+x file   # 给文件所有者增加执行权限
chmod go-w file  # 去掉组和其他用户的写权限
```

用八进制法表示：

```sh
chmod 755 file
```

| 数字 | 二进制 | 权限 |
| ---- | ------ | ---- |
| 7    | 111    | rwx  |
| 6    | 110    | rw-  |
| 5    | 101    | r-x  |
| 4    | 100    | r--  |
| 3    | 011    | -wx  |
| 2    | 010    | -w-  |
| 1    | 001    | --x  |
| 0    | 000    | ---  |

第一个是所有者，第二个是组用户，第三个是其余用户。

**滥用软链接攻击（symlink attack）**

用户可能创建一个指向关键系统文件的软链接，例如：

```sh
ln -s /etc/passwd ~/fake
chmod 777 ~/fake  # 对象其实是 /etc/passwd！
```
